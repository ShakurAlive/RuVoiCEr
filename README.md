# RuVoiCer

**Локальный перевод аудио EN → RU с сохранением голоса, тембра и интонации.**

Полностью офлайн-пайплайн на базе open-source моделей.  
Оптимизирован под **Windows 11 + RTX 4070 Super (12 GB VRAM)**.

---

## Как это работает

```
Аудио (EN)
  │
  ├──► Demucs (разделение голос / фон)
  │       ├── vocals.wav      ──► Pyannote (диаризация спикеров)
  │       │                          └──► Whisper ASR ──► Текст (EN)
  │       │                                                  └──► LLM / NLLB перевод ──► Текст (RU)
  │       │                                                                                  └──► CosyVoice 3.0 ──► Речь (RU)
  │       └── background.wav  ──────────────────────────────────────────────────────────────────────┐
  │                                                                                                 ▼
  └────────────────────────────────────────────────────────────────────────────── Микширование ──► Аудио (RU)
```

| Шаг | Модель | Назначение |
|-----|--------|-----------|
| 0. Разделение аудио | **Demucs htdemucs** (Meta) | Разделение на чистый голос и фон (музыка, SFX) |
| 1. Диаризация | **Pyannote** speaker-diarization-3.1 | Определение спикеров и извлечение референсов |
| 2. Распознавание речи | **faster-whisper** (large-v3-turbo) | Английский аудио → текст + таймкоды |
| 3. Перевод | **Ollama** (qwen2.5:14b) или **NLLB-200** (3.3B) | Естественный перевод на русский |
| 4. Синтез речи | **CosyVoice 3.0** (Fun-CosyVoice3-0.5B) | Озвучка русского текста голосом оригинала |
| 5. Сборка | **pydub** / FFmpeg | Склейка сегментов с паузами + микширование с оригинальным фоном |

> Модели загружаются **последовательно** (load → use → unload), чтобы не превышать 12 GB VRAM.

## Быстрый старт

### 1. Предварительные требования

- **Python 3.10+** — [python.org](https://python.org)
- **CUDA 12.x** — поставляется с драйверами NVIDIA
- **FFmpeg** (рекомендуется) — [gyan.dev/ffmpeg](https://www.gyan.dev/ffmpeg/builds/)
- **Git** (для клонирования CosyVoice) — [git-scm.com](https://git-scm.com)
- **Ollama** (для лучшего качества перевода) — [ollama.com](https://ollama.com)

### 2. Установка

```bat
install.bat
```

Скрипт создаст виртуальное окружение, установит PyTorch с CUDA и все зависимости.

### 3. Скачивание модели CosyVoice 3.0

```bat
:: Клонируем CosyVoice (и сабмодуль Matcha-TTS)
git clone --recursive https://github.com/FunAudioLLM/CosyVoice.git third_party/CosyVoice

:: Скачиваем модель (~7 GB)
pip install modelscope
python -c "from modelscope import snapshot_download; snapshot_download('iic/CosyVoice3-0.5B', local_dir='pretrained_models/Fun-CosyVoice3-0.5B')"
```

### 4. Подготовка Ollama (рекомендуется)

```bat
ollama pull qwen2.5:14b
ollama serve
```

> Если Ollama не нужен, в интерфейсе можно выбрать **NLLB-200** — полностью офлайн-перевод.

### 5. Запуск

```bat
run.bat
```

Откроется браузер с GUI на http://127.0.0.1:7860.

---

## Интерфейс

1. **Загрузите аудиофайл** (английская речь).
2. *(Опционально)* Загрузите **эталон голоса** — 5–30 секунд чистой речи спикера.  
   Если не указан, эталон извлекается автоматически из чистого вокала (после Demucs).
3. Настройте параметры (модель Whisper, метод перевода).
4. Нажмите **«Запустить обработку»**.
5. Слушайте результат и скачивайте готовый WAV.

## Ожидаемое качество

| Параметр | Оценка |
|----------|--------|
| Тембр голоса | ~97 % совпадение (CosyVoice 3.0 in-context learning) |
| Интонация / эмоции | 90–95 % (SOTA cross-lingual prosody) |
| Фоновые звуки | Сохраняются через Demucs + микширование |
| Точность перевода | Высокая (Ollama) / хорошая (NLLB) |
| Скорость | ~30–90 мин на 1 час аудио (RTX 4070 Super) |

---

## Советы для лучшего результата

- **Фоновые звуки**: Demucs автоматически отделяет голос от музыки/SFX. Фон микшируется в результат.
- **Эталон голоса**: автоматически извлекается из чистого вокала (после Demucs) — без шумов и музыки.
- **Несколько спикеров**: Pyannote определяет до 10+ спикеров. Каждый получает свой референс для CosyVoice.
- **Перевод**: Ollama с qwen2.5:14b даёт более литературный перевод, чем NLLB.
- **Длинные файлы**: Whisper автоматически разбивает на сегменты по паузам — это нормально.
- **VRAM**: Модели загружаются последовательно (load → use → unload). Если не хватает памяти, уменьшите модель Whisper (medium / small).
